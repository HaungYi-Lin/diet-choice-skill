# AI 輔助外食減脂判斷 Skill

## 專案介紹

本 Skill 是一個用於協助日常外食選擇的 AI 工作流程，主要目標是幫助使用者在減脂期間判斷某項食物是否適合食用。

透過固定的分析步驟，AI 會依照食物類型、熱量風險、蛋白質、油脂、糖分、飽足感與用餐情境進行判斷，最後輸出推薦程度、原因分析與替代建議。

## 使用情境

本 Skill 適合用於以下情境：

- 不確定某個外食是否適合減脂
- 想知道某餐怎麼吃比較不容易超標
- 想比較不同餐點的推薦程度
- 想獲得更適合減脂的替代選擇
- 想快速整理飲食選擇的判斷邏輯

## Skill 目標

本 Skill 的目標不是禁止使用者吃某些食物，而是協助使用者做出更有意識的飲食選擇。

AI 會根據輸入內容，提供：

1. 推薦程度
2. 食物類型判斷
3. 熱量與營養風險分析
4. 是否適合減脂期間食用
5. 改善吃法
6. 替代建議
7. 一句話總結

## 輸入格式

使用者可以輸入以下資訊：

- 食物名稱
- 用餐時間
- 目前目標，例如減脂、維持、增肌
- 是否有運動
- 是否已經吃過其他高熱量食物
- 飲料選擇
- 其他補充條件

範例：

```text
食物：雞排便當
目標：減脂
用餐時間：午餐
飲料：無糖綠茶
今天沒有運動
