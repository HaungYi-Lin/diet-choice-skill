# AI 輔助外食減脂判斷 Skill

## 專案介紹

本 Skill 是一個用於協助日常外食選擇的 AI 工作流程，主要目標是幫助使用者在減脂期間判斷某項食物是否適合食用。

透過固定的分析步驟，AI 會依照食物類型、熱量風險、蛋白質、油脂、糖分、飽足感與用餐情境進行判斷，最後輸出推薦程度、原因分析、改善吃法與替代建議。

本專案除了整理 Skill 流程，也建立可直接使用的 GPT 助手，使用者可以輸入想吃的食物、飲料或餐點，讓 GPT 依照固定規則進行分析。

## 使用情境

本 Skill 適合用於以下情境：

- 不確定某個外食是否適合減脂
- 想知道某餐怎麼吃比較不容易超標
- 想比較不同餐點的推薦程度
- 想獲得更適合減脂的替代選擇
- 想快速整理飲食選擇的判斷邏輯
- 想在早餐、午餐、晚餐中做出較適合減脂的外食選擇

## Skill 目標

本 Skill 的目標不是禁止使用者吃某些食物，而是協助使用者做出更有意識的飲食選擇。

AI 會根據輸入內容，提供：

1. 推薦程度
2. 食物類型判斷
3. 熱量與營養風險分析
4. 是否適合減脂期間食用
5. 改善吃法
6. 替代建議
7. 一句話總結

## 輸入格式

使用者可以輸入以下資訊：

- 食物名稱
- 用餐時間
- 目前目標，例如減脂、維持、增肌
- 是否有運動
- 是否已經吃過其他高熱量食物
- 飲料選擇
- 其他補充條件

範例：

```text
食物：雞排便當
目標：減脂
用餐時間：午餐
飲料：無糖綠茶
今天沒有運動
```

## 如何使用本 GPT Skill

本 Skill 已製作成可直接使用的 GPT 助手。

使用方式：

1. 開啟「AI 外食減脂判斷助手」
2. 輸入想分析的食物、飲料或餐點
3. 例如：
   - 雞排便當可以吃嗎？
   - 早餐店蛋餅加奶茶適合減脂嗎？
   - 鍋燒烏龍適合減脂嗎？
   - 吃到飽要怎麼吃比較不容易爆熱量？
   - 麥當勞套餐減脂期間怎麼選？
4. GPT 會依照外食減脂判斷流程，分析食物類型、熱量風險、蛋白質、油脂糖分與飽足感
5. 最後輸出推薦程度、原因分析、改善吃法與替代選擇

## 對話啟動器設計

本 GPT 設計了早餐、午餐、晚餐等常用啟動問題，方便使用者快速取得建議。

建議對話啟動器：

```text
幫我推薦減脂早餐
幫我推薦減脂午餐
幫我推薦減脂晚餐
幫我安排今天三餐外食建議
```

當使用者點選早餐、午餐或晚餐建議時，GPT 會直接依照一般外食減脂情境提供建議，不需要使用者先輸入完整資料。
